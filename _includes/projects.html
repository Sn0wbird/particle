Ninja Chat

My Ninja-Chat App uses the capabilities of the Firestore, creating a real-time app in the front end in Vue and hooking it up to a Vue Instance. The listener in our front end Vue App listens to any changes and pulls down the updated data automatically without us having to reload the page. Real-time data updates are a central feature of this app, allowing users to chat with each other. For this project I used JavaScript in the Vue JS framework, Materialize CSS, and the FireStore database.

URL: chat.sn0wbird.com

Ninja Smoothies

My Ninja-Smoothie App allows the user to create, edit and delete smoothie recipes! Through creating this app, I learnt how to store data in the database in Firebase. Creating this app introduced to me the concept of C.R.U.D, (create, read, update and delete) For this project I used JavaScript in the Vue JS framework, Materialize CSS, and the FireStore database.

URL: smoothies.sn0wbird.com
